# Minecraft Clone
A 3D game dev project. This engine is just for experimental purposes and is being used to make a Minecraft clone to learn C++, SDL3, GLAD, and GLM.

SDL3 Version: 3.2.24

SDL3 TTF Version: 3.1.0

SDL3 Image Version: 3.2.4

GLM Version: 1.0.2

GLAD Version: 4.6 (Language: C/C++, Specification: OpenGL, API: gl, Profile: Core, Extensions: Default)
